# conics.js

A tool to simplify and identify conics from their equations.

## Getting Started

### Prerequisites

> [Node](https://nodejs.org)

### Installing

Clone the repository

```
git clone https://github.com/pedromartelleto/simath.js.git
```

Install dependencies

```
npm install
```

Run

```
npm start
```

## Running the tests

```
npm test
```
